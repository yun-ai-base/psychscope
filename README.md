# PsychScope — 心理学大师知识图谱

一个精心策展的中外著名心理学家知识网站，收录人类历史上最具影响力的15位心理学大师。

## 包含内容

- **生平事迹**与学术生涯
- **经典著作**及创作背景、核心思想
- **代表言论**与名言
- **学说发展脉络**与流派演变
- **社会影响**与学术评价
- **同行传承**与理论突破
- **社会应用**与实际价值
- **AI 智评**（原创性、影响力、实用性三维评分）
- **学派分类**（精神分析、行为主义、人本主义、认知、积极心理学等）
- **经典实验**专栏
- **相关拓展**资源（博物馆、传记、纪录片、课程等，含真实链接）

## 技术栈

纯原生 HTML5 + CSS3 + JavaScript，零框架零依赖，适配移动端。

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何构建工具或服务器。
